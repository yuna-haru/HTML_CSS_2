Flexレイアウト

レイアウトに柔軟性があるなど、旧レイアウトの欠点を解消し、利点が多い。
HTMLにて、

＜div class="flex"＞
＜div＞1＜/div＞
＜div＞2＜/div＞
＜div＞3＜/div＞
＜div＞4＜/div＞
＜/div＞
と記載、次にCSSにて

.flex { background-color: #00c7ce; height: 50vh; padding: 10px; display: flex; }
.flex > div { background-color: #fff; margin: 5px; font-size: 30px; }
と記載する。

Flexコンテナ―

Flexレイアウトを行うための親の箱です。
上記の .flex にdisplay:flexと設定することでFlexコンテナーになる
Flexコンテナーは並ぶ方向(横・縦) / 横方向の整列方法 / 縦方向の整列方法 / 折り返し方法
等を同時に設定できる

Flexアイテム

Flexコンテナー内に直接入っている要素は全て(※)自動的にFlexアイテムとなります。
Flexアイテムとなった要素は更に 並び順 / 伸縮比率 / 個別の縦位置など
簡単に設定することができます。
※Flexアイテム内に更にdivが含まれるなど、入れ子になっているアイテムはFlexアイテムとはなりません

Flexコンテナーで指定できるプロパティ

・flex-direction
縦並び、横並びの指定ができる。
rowが横並び、columnが縦並びになる。
rowやcolumnのうしろに-reverseを記述すると並び順が逆になる

・flex-wrap
折り返しの指定ができる。
no-wrapは初期値、wrapは左上から折り返す、wrap-reverseは左下から折り返す。

・flex-direction: row;
flex-directionとflex-wrapをまとめて記載(※以下、例)
.flex { display: flex; flex-flow: row wrap; }

・justify-content
Flexコンテナーの内容の横方向の整列方法の指定
flex-startは左寄せ、flex-endは右寄せ、centerは真ん中
space-betweeenは各アイテムを均等に配置し最初のアイテムは先頭に寄せ、最後のアイテムは末尾に寄せる
space-aroundは各アイテムを均等に配置し各アイテムの両側に半分の大きさの間隔を置く
space-evenlyは各アイテムを均等に配置し各アイテムの周りに同じ大きさの間隔を置く
stretchは各アイテムを均等に配置しサイズが 'auto' であるアイテムは、コンテナーに合わせて引き伸ばす

・align-items
縦方向の整列方法の指定
stretchは初期値、flex-startはflexアイテムを先頭寄せ、flex-endはflexアイテムを末尾寄せ。
centerはアイテムの中央寄せ、baselineはアイテムの内容を揃える。

・align-content
Flexコンテナーの内容が複数行になった際の整列方法
内容はjustify-contentと同じ。

Flexアイテムで指定できるプロパティ

・order
Flexアイテムの順序の指定。
いちいち入れ替えなくとも疑似クラス(nth-child(n))を使って順序が変えられる

・flex-grow
Flexアイテムの横幅の伸びる比率の指定
疑似クラスで0から1、2、3……と伸びる長さを変えられる

・flex-shrink
Flexアイテムの横幅の縮む比率の指定。
疑似クラスで0から1、2、3……と縮む大きさを変えられる

・flex-basis
Flexアイテムの横幅の指定。
widthでも問題ないが、
widthとの大きな違いは、親の箱（Flexコンテナー）のflex-directionが縦並び（columnまたはcolumn-reverse)
であれば自動で高さの設定に切り替わるという点です

Flex

flex-grow, flex-shrink, flex-basis をまとめて指定できます。
値の指定方法や個数によって表示のされ方が変わるため、注意が必要です。
.flex > div {
・単位がない数値を1つ指定 → flex-growに
flex: 2;
・単位のある数値を1つ指定 → flex-basisに
flex: 10em;
flex: 30%;
・単位がない数値を2つ指定 → flex-grow | flex-shrink に
flex: 2 2;
・値を3つ指定する → 必ず flex-grow | flex-shrink | flex-basis に
flex: 2 2 10%;
}

・align-self
Flexアイテムの縦方向の位置を、アイテムごとに個別に指定
flex-startはflexアイテムを起点に配置(一番上を起点)
flex-endはflexアイテムの末尾を起点に配置(一番下を起点)
centerはflexアイテムの中央に配置
stretchは寸法が 'auto' のアイテムをコンテナーに合うよう伸長
